# Trabalho-Prog3-Java
Trabalho para matéria Programação III

Dentro do diretório *src*:

### Para construir os arquivos executáveis: 
```
ant compile
```

### Para rodar:
#### Read e Write
```
ant run
```
### Read only
```
ant run-read-only
```
### Write only
```
ant run-write-only
```

### Para limpar .class:
```
ant clean
```
